# ruby-getIndexOfSubtext
An algorithm to match a substring to a string and return the substring index

Version: ruby 1.9.3

1. Search for 'Start Command prompt with Ruby' and launch it.

2. Access the file path directly from the command line or drag .rb file into the command line and hit enter

3. Enter a sentence when prompted and hit enter i.e 'she sells sea shells on the sea shore'

4. Now enter matching text and hit enter i.e 'sea', 'e', 'q'

5. Indexes of all matching texts are shown as output. If there is no match, the output will display ' No matches' to the user
